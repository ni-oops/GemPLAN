**GemPLAN** – локальный веб-планировщик для визуального управления задачами. Приложение позволяет группировать дела по категориям с индивидуальной цветовой кодировкой и иконками для быстрой навигации.

**Инструкция**
1. Создание категории: Введите название раздела (например, «Работа» или «Личное»), выберите цвет и иконку (по-умолчанию назначаются автоматические).
2. Добавление задачи: Введите текст задачи в соответствующей категории.
3. Управление: Отмечайте задачи как выполненные кликом или удаляйте записи, чтобы поддерживать список актуальным.

**Данные и конфиденциальность**
Приложение работает без регистрации и облачных серверов – вся информация хранится исключительно в локальной памяти вашего браузера. Ваши данные полностью приватны.

**Перенос данных**
Для работы на разных устройствах (синххронизации) используйте кнопки «Скачать» (экспорт списка в файл) и «Загрузить» (восстановление данных из файла).

Примечание: Данные привязаны к конкретному браузеру. Очистка кэша или открытие приложения в другом браузере приведет к сбросу настроек, поэтому регулярно делайте экспорт текущего состояния с помощью кнопки «Скачать».
